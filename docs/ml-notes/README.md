# Notas de Machine Learning

## Origen
Estas notas provienen del curso **"Econometría en la Era de las Máquinas"** y resúmenes temáticos de ciencia de datos aplicada. Fueron destiladas de un repositorio central de notas y seleccionadas por relevancia para este proyecto.

## Fuentes Originales
- **data-science-notes**: Curso completo de econometría + ML (5 clases, 9 resúmenes temáticos, 2 cheatsheets)
- **data-science-notes_latest_only**: Guías de implementación y producción de árboles de decisión

## Estructura
- `cheatsheets/` — Checklist de modelado y cheatsheet de econometría/ML
- `resumos/` — Resúmenes temáticos (bias-variance, métricas, etc.)
- `cursos/` — Clases del curso de econometría
- `implementaciones/` — Guías de implementación con código Python
- `produccion/` — Guías de hardening para producción

## Cómo Usar
1. Consulta `MachineLearning.MD` en la raíz del proyecto para contexto específico
2. Usa los cheatsheets como referencia rápida durante desarrollo
3. Revisa los resúmenes temáticos para profundizar en conceptos

## Cómo Actualizar
Estas notas son copias autónomas (no symlinks). Para actualizar:
1. Verificar cambios en el repositorio fuente: `data-science-notes/`
2. Copiar archivos actualizados manualmente
3. Actualizar la fecha en este README
